# JustMySocksBandwidth

## Description

Present bandwidth of JustMySocks visually

## USAGE

1. Get `UUID` from JMS `Bandwidth counter API link`
2. Import the script into `scriptable`
3. Add widget on homepage with parameter `UUID`
